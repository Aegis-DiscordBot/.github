## 👋 Welcome to the Aegis Organization

This is the official GitHub home of **Aegis** — a powerful, all-in-one Discord bot built for server administrators who take their community seriously.

We build fast, reliable, and beautifully designed tools for Discord communities of all sizes. Whether you're running a small friend group or a large public server, Aegis has you covered.

---

## ✨ What Aegis Does

- 🔨 **Moderation** — Ban, kick, warn, mute & timeout with full audit trails
- 🛡️ **Anti-Nuke** — Detects and stops server raids, mass bans, and permission abuse in real time
- 🤖 **Automod** — Smart spam, link, and bad word filtering out of the box
- 📋 **Logging** — Detailed logs for every action taken in your server
- 🎵 **Music** — High quality music playback with queue management
- 📣 **YouTube Alerts** — Get notified when your favourite creators post or go live
- 🖥️ **Dashboard** — A sleek, futuristic web UI to configure everything without a single command

---

## 🛠️ Our Tech Stack

We build with modern, fast, and type-safe technologies:

- **TypeScript** — Type safety across the entire codebase
- **Bun** — Blazing fast JavaScript runtime
- **Seyfert** — Modern Discord bot framework
- **PostgreSQL + Drizzle ORM** — Reliable, structured data storage
- **Next.js** — Fast, beautiful web dashboard
